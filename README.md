## How to Use This Curriculum

### Daily Routine
1. **Read** the lesson content (20-30 min)
2. **Code** the exercises (60-90 min)
3. **Reflect** and answer comprehension questions (10-15 min)
4. **Commit** your work to GitHub

### Lesson Structure
Each lesson contains:
- `README.md` - Theory and concepts
- `exercises/` - Hands-on coding tasks
- `solutions/` - Reference solutions (try first!)
- `quiz.md` - Comprehension questions

### Progress Tracking
- [ ] Complete each lesson's exercises
- [ ] Pass the quiz (answer all questions correctly)
- [ ] Complete the module project
- [ ] Push to GitHub with meaningful commits

### GitHub Portfolio Structure
```
mlops-agentic-ai-curriculum/
├── README.md (this file)
├── progress.md (your progress tracker)
├── 01-python-for-mlops/
│   ├── lesson-1.1-data-structures/

│   └── project/
└── ...
```
